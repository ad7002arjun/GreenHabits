# 🌱 GreenHabits

A full-stack eco-friendly habit tracking application built with MERN stack.

## Tech Stack
- **Frontend:** React.js, React Router, Context API, Axios
- **Backend:** Node.js, Express.js, JWT, Bcrypt
- **Database:** MongoDB Atlas, Mongoose

## Features
- 🔐 Secure authentication (JWT + Bcrypt)
- 🌿 Track eco-friendly habits across 6 categories
- 📊 Measure real environmental impact
- 🏆 Gamification with levels and streaks
- 📈 Visual analytics and weekly trends
- 🌍 Real-world impact equivalents

## Modules
1. Dashboard - Overview and stats
2. Habit Management - Add/delete eco habits
3. Habit Logging - Record daily actions
4. Impact Dashboard - Visualize environmental impact
